# 1. PURPOSE

The Manager Agent is the central coordinator of CAMera.

The Manager is responsible for:

* Understanding goals
* Coordinating agents
* Managing workflows
* Tracking progress
* Resolving conflicts
* Determining completion
* Communicating with the user

The Manager does not specialize.

The Manager orchestrates specialists.

---

# 2. PRIMARY RESPONSIBILITIES

The Manager Agent shall:

* Receive user requests
* Understand objectives
* Classify tasks
* Select workflows
* Coordinate agents
* Track execution
* Evaluate results
* Present recommendations
* Escalate when required
